  YD_BIM 工具 v2.2.9 - Revit 外掛
═══════════════════════════════════════════════════════════════

感謝您選擇 YD_BIM 工具！

本外掛為 Autodesk Revit 提供了一套完整的 BIM 工具集，
包含 AR 工具、資料管理工具和 COBie 匯入匯出功能。

───────────────────────────────────────────────────────────────
  系統需求
───────────────────────────────────────────────────────────────

• Autodesk Revit 2024 / 2025 / 2026（任一版本）
• Windows 10/11 (64-bit)
• .NET Framework 4.8 或更高版本
• 管理員權限（用於安裝）

───────────────────────────────────────────────────────────────
  功能模組
───────────────────────────────────────────────────────────────

【關於面板】
  • 授權管理 - 管理軟體授權
  • 關於 YD_BIM 工具 - 查看版本資訊

【AR 面板】
  ▸ 模板工具組（5 個工具）
    • 模板生成 - 自動生成模板
    • 刪除模板 - 批次刪除模板
    • 面選模板 - 從面選擇生成模板
    • 匯出CSV - 匯出模板資料
    • 結構分析 - 結構分析工具
    
  ▸ 裝修工具組（1 個工具）
    • 裝修生成 - 自動生成裝修元件
    
  ▸ 接合工具組（2 個工具）
    • 自動接合 - 自動接合結構元件
    • 接合到選取 - 將元件接合到選取的目標

【資料面板】
  • 欄位管理 - 管理 COBie 欄位
  • 範本匯出 - 匯出 COBie 範本
  • COBie匯出 - 匯出 COBie 資料
  • COBie匯入 - 匯入 COBie 資料

───────────────────────────────────────────────────────────────
  安裝說明
───────────────────────────────────────────────────────────────

1. 關閉所有 Revit 應用程式
2. 執行安裝程式 YD_BIM_Tools_v2.2.9_Setup.exe
3. 安裝程式會自動偵測您電腦上已安裝的 Revit 版本
4. 選擇您要安裝外掛的 Revit 版本（可多選）
   • Revit 2024
   • Revit 2025
   • Revit 2026
5. 依照安裝精靈的指示完成安裝
6. 重新啟動 Revit
7. 在 Ribbon 介面中找到「YD_BIM」標籤

注意：只有已安裝的 Revit 版本才會顯示在選項中。

───────────────────────────────────────────────────────────────
  解除安裝
───────────────────────────────────────────────────────────────

方法 1: 使用 Windows 控制台
  • 開啟「控制台」→「程式和功能」
  • 找到「YD_BIM 工具」
  • 點擊「解除安裝」

方法 2: 使用開始選單
  • 開啟「開始選單」
  • 找到「YD_BIM 工具」資料夾
  • 點擊「解除安裝 YD_BIM 工具」

───────────────────────────────────────────────────────────────
  常見問題
───────────────────────────────────────────────────────────────

Q: 安裝後在 Revit 中看不到 YD_BIM 標籤？
A: 請確認：
   1. Revit 已完全重新啟動
   2. 檢查 Revit 的「外部工具」→「外掛管理員」
   3. 確認沒有錯誤訊息

Q: 圖示沒有顯示？
A: 這通常是因為：
   1. 圖示檔案未正確安裝
   2. 請嘗試重新安裝

Q: 出現授權錯誤？
A: 請聯繫技術支援取得授權
